# 🌍 Earthquake Data Wrangling Project Using SQL (MySQL)

## 📌 Project Overview

This project focuses on cleaning and transforming raw earthquake data using SQL in MySQL Workbench.

The dataset contains earthquake records including date, time, magnitude, depth, and location information.

The main objective of this project is to perform data wrangling and prepare the dataset for analysis and visualization.
